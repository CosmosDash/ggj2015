# ggj2015
global game jam 2015

you need python and pygame if you want to run our game :

for Windows :


Vous devez posséder une version 32 bits de Python pour que Pygame 
fonctionne.

Si vous avez installé la version 64 bits, désinstallez-la et 
réinstallez la version 32 bits.

goto http://pygame.org/download.shtml

Choisissez le fichier qui correspond à votre version de Python 
dans la liste "Windows".

Téléchargez le fichier msi qui contient tout ce dont vous 
avez besoin pour coder avec Pygame, et lancez-le !

Pygame devrait repérer Python sur votre machine.

Si ce n'est pas le cas, vous aurez seulement 
la seconde ligne, précisez alors le répertoire 
d'installation de Python sur votre machine.

Pour vérifier que l'installation a bien 
été effectuée, vous pouvez vous rendre dans 
l'interpréteur Python et taper : 
import pygame

S'il n'y a pas d'erreur c'est bon, vous pouvez lancez game.py
