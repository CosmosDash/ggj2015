# ggj2015
global game jam 2015

you need python2.7 and pygame if you want to run our game

for Unix system:
    python and pygame are probably in you package system (apt-get, pacman, portage, whatever)
    (if you encounter any problem, go to http://pygame.org/download.shtml)

for Windows :
You need to have a 32-bits version of python to run pygame.
If you have a 64-bits one, uninstall it and re-install the 32bits one.

goto http://pygame.org/download.shtml

Choose the file corresponding to your python version in "Windows"

Download .msi file containing everything you need to code with pygame, and run it !

If Pygame don't find python on your computer, just specify the repertory in which you installed it.

You can now try tu run game.py

Vous avez besoin de python2.7 et de pygame si vous voulez jouer a notre jeu

Sur les systemes Unix:
    python2.7 et pygame sont probablement dans votre gestionaire de paquets.
    (en cas de probleme allez sur http://pygame.org/download.shtml)

Sur Windows:
Vous devez posséder une version 32 bits de Python pour que Pygame 
fonctionne.

Si vous avez installé la version 64 bits, désinstallez-la et 
réinstallez la version 32 bits.

goto http://pygame.org/download.shtml

Choisissez le fichier qui correspond à votre version de Python 
dans la liste "Windows".

Téléchargez le fichier msi qui contient tout ce dont vous 
avez besoin pour coder avec Pygame, et lancez-le !

Pygame devrait repérer Python sur votre machine.

Si ce n'est pas le cas, vous aurez seulement 
la seconde ligne, précisez alors le répertoire 
d'installation de Python sur votre machine.

Pour vérifier que l'installation a bien 
été effectuée, vous pouvez vous rendre dans 
l'interpréteur Python et taper : 
import pygame

S'il n'y a pas d'erreur c'est bon, vous pouvez lancez game.py
